SOBRE AS OPÇÕES NO MENU:

[1] Extrai o URL de sites como http e https, esta opção faz a busca de subdomínios... Digite o URL de sua operadora ou algum site global e encontre os subdomínios dele

[2] Mostra o status do site encontrado... É um Host Checker que vai lhe ajudar a encontrar as hosts válidas com o status 200. Mas atenção! A opção 2 só funciona se você salvar alguma host na opção 3.

[3] Um bloco para adicionar sites e poder fazer a checagem na opção 2... Você pode buscar os subdomínios na opção 1, copiar, vir na opção 3, salvar e usar a opção 2 para encontrar as que estão com status 200... Salve as hosts em lista:

Exemplo:

google.com
github.com
youtube.com

Após a lista, clique CTRL + C para sair e salvar, então use "bash BUSCADOR.sh" para abrir o script novamente

[4] Gere algumas payloads com a host 200 que você encontrou (algumas payloads geradas precisam ser editadas!)

[5] Mostrar as portas que estão abertas no URL e mais algumas informações

[6] Mostrar o PROXY IP da Host e mais algumas informações

[7] Rastreie o IP... Encontre a localização do PROXY encontrado, com localização e mais algumas informações

[8] Leia o manual de uso do Script com atenção

[9] Créditos ao criador do script, link da conta no telegram, link do Bot para arquivos, tutoriais e configurações e uma mensagem do criador do Script

[0] Sair do script 


Clique em ENTER para voltar ao menu inicial...
